## 依頼項目 (Codex 神回 34)

### A. ワークフロー選択 — 優先度付きランキング

以下の候補について、(コイケ向け本案件で) 採用優先度を ★ 5 段階で評価:

| # | 案 | 概要 | コスト/時間 |
|---|---|---|---|
| 現状 | html2pptx (公式 skill 推奨) | HTML 24 枚 → Playwright → pptx | 60-90 分、CSS/HTML 書く |
| A | python-pptx + 既存テンプレ (.pptx) | テンプレ読込 → slide.layout 活用 + 内容差替 | テンプレ用意必要 (なければゼロから) |
| B | Marp + Markdown | `marp slides.md -o out.pptx` で軽量 | 動画埋込・凝ったレイアウトに弱い |
| C | Gamma / Tome / Pitch (Web AI) | プロンプトで AI 生成 → pptx export | ブランドコントロール弱、有料 |
| D | Claude for PowerPoint アドイン | PowerPoint 内で Claude 起動、対話的に作成 | 環境依存、Windows + Office 365 必要 |
| E | PowerPoint テンプレート (.potx) + python-pptx 差替 | プロが作った .potx に CC が内容差し込み | .potx を別途調達 (購入 / Microsoft 公式テンプレ) |
| F | HTML → PNG (1 枚画像化) → pptx に画像埋込 | デザインは HTML/CSS、品質保証は「画像なので崩れない」 | 編集不可になる、文字検索不可、容量大 |
| その他 | Codex お勧めワークフロー (具体的に) | - | - |

特に重視する観点:
- ✅ **5/21 までに間に合う**
- ✅ **経営層向けに恥ずかしくない品質** (文字溢れ・配置崩れゼロ)
- ✅ **後日動画埋め込みが容易**
- ✅ **柏原が PowerPoint で開いて微修正できる** (本番直前に文言調整等)

### B. 時間効率 vs 品質 トレードオフ判断

選択肢:
1. **v3 を最後まで仕上げる** (html2pptx ワークフロー、60-90 分)
2. **別ワークフローに切り替え** (案 A-F のいずれか、所要時間予測も)
3. **v2 を磨いて完成版に** (python-pptx で文字溢れだけ直す、30-60 分)
4. **柏原が PowerPoint で v2 を手修正** (CC ワーク完了、柏原が 1-2 時間で仕上げ)
5. **既存パンフ + portal スクショの組合せ** (CC 生成スライドを最小化、柏原が PowerPoint で組立)
6. その他

### C. 動画埋め込みのタイミング判断

選択肢:
1. **CC で pptx に動画埋込 (add_movie)** — 公式 pptx skill 推奨だが experimental API
2. **柏原が PowerPoint 当日に手動挿入** — 確実、柏原作業 30 分
3. **動画なし pptx + 別ウィンドウで動画再生** — pptx 化を簡素化、当日 OS 操作で動画再生
4. **動画スライドだけ別 pptx に分けて統合** — トラブル時の切り戻し容易

特に Slide 07 (ダイヘン AiTran) は会場の実演セミナーへの誘導動線になるため、必須レベル。
他 3 本 (OTOS ×2 / オプティ) は補強訴求。

### D. コイケ酸商経営層向けに本当に効くデザイン

コイケ酸商 = 中堅商社、元倉社長 + 常務 + 各出先責任者。
業界: 溶接・ガス・産業材専門商社。

候補スタイル:
- **McKinsey 風** (シャープ + 青/ネイビー、洋風コンサル) ← v2 で試した
- **Apple 風** (ミニマル + 大量の余白、テック系) 
- **Stripe 風** (ダーク + パープルグラデ、開発者向け)
- **日本の上場企業 IR 資料風** (堅実、グラフ多用、保守的、紺・グレー基調)
- **メーカー営業資料風** (現場感、写真多用、製品スペック表)
- **TWF portal と統一感** (portal オレンジ #F97316 + ネイビー、本案件のブランド)

「中堅商社の経営層に派手すぎないが品があるトーン」とは具体的にどんなデザイン要素か?
