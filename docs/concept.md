# TWF2026 みどころポータル コンセプト

## 何を作るか
マツモト産業㈱京葉営業所が主催店向けに公開する、TWF2026専用の**営業支援ポータルサイト**。
公式サイト（mac-exe.co.jp/welding/welding_new/tokyo/）とは別物の、外部公開URL自前サイト。

## ターゲット
京葉営業所の**主催店（外部取引先）**。エンドユーザー向けではない。

## 真の目的
「みどころ紹介サイト」ではない。**主催店の売上に繋げる3軸の営業支援ツール**。

| シーン | 提供価値 |
|---|---|
| **TWF前**: 顧客誘致 | メーカー企画の刺さるポイント、特典内容、実演有無 |
| **TWF当日**: 顧客同行 | 効率的な導線、必見リスト、ブース番号 |
| **TWF後**: 商談化 | 問い合わせ先、関連カタログ、後追い資料 |

すべての設計判断は「主催店の売上に効くか」を基準にする。

## 公式サイトとの住み分け

| | mac-exe.co.jp 公式 | 本サイト |
|---|---|---|
| 対象 | エンドユーザー | 京葉の主催店 |
| 内容 | 「TWF行きましょう」案内 | 「TWF活用して売りましょう」ツール |
| 役割 | 集客 | 営業支援 |
| 更新主体 | 担当部署 | 京葉2課（柏原） |
| 機能 | 来場登録、PDF配布、テキスト案内 | 検索/フィルタ/詳細/特典バッジ |
| 印象 | 招待制案内 | プロ仕様の営業ツール |

完全に補完関係。被らない。フッターに公式サイトへのリンクを入れて関係を明示する。

## 設計哲学
**素材 > レイアウト**

モック画像を直接実装してもAI感が抜けない。
GPT-Image-2でグリッド形式で**素材だけ抽出**し、1個ずつ分割・透過してから自由配置する。
これがプロのデザイナーの発想で、AI感を消す唯一の方法。
Phase 3-4の作り込みは、この前提で進める。

## サイト構造

### TOP
- カウントダウン（リアルタイム秒単位、TWF=2026/6/12 10:00開幕）
- 主催店の使い方ガイド（3軸提示）
- 注目企画ピックアップ（柏原セレクト5-10案件）
- メーカー一覧（検索/フィルタ/カテゴリ、各カードに「提案フック」「実演有無」「特典」）
- **当日セール特価**（TWF事務局配布の `26TWF_sale.pdf` をベースに、当日限定価格・成約特典・抽選会情報を強調表示）
- 当日チェックリスト印刷

> **Section 5 方針変更メモ（Phase 5）**
> 当初は「業界別推奨ルートマップ」を置く予定だったが、以下の理由で「当日セール特価」に変更:
> - 公式の会場マップは TWF事務局から配布済み（主催店として既に取得済み）。マツモト京葉として独自に作る必要なし
> - 価格情報の方が主催店の営業ツールとして直接的に売上に効く
> - マツモトの京葉営業所としての独自価値は **「客の課題理解 × 適切なメーカー紹介」**。マップの再発明ではない
> - 参照素材: `assets/raw/_official/26TWF_sale.pdf` / `26TWF_pamphlet.pdf`

### メーカー詳細 `/m/{slug}/`
- ブース番号、カテゴリ
- 企画概要
- 新製品・新技術
- ブースのみどころ

## データソース

```
\\flsv04\...\回答集約\
  ├ TWF2026_回答集約.xlsx       ← メーカー回答自動集約（10分間隔更新）
  └ attachments\{メーカー名}\   ← 各社の配布資料PDF・画像
```

メーカー回答が薄い社（「特にありません」が多い）は `data/柏原追記.xlsx` で補完。
表示優先度: **柏原追記 > メーカー回答 > 「展示予定」**

## 運用方針

- **更新**: 集約Excel変更→朝1回自動デプロイ（Windowsタスクスケジューラ）
- **段階公開**: 5月初旬に40-50社で先行URL配布、以降随時更新
- **NEWバッジ**: 新規追加メーカーは1週間表示
- **承認**: 天満支店長承認済み
- **主催店リスト**: 不要（こちらからURL直送）
- **ホスティング**: Cloudflare Pages（pages.devで十分）
- **認証**: 共通パスワード方式が候補（未確定）

## スコープ外（やらない判断）

- 本格的な認証システム（URL限定+共通パスで十分）
- メーカーロゴ画像生成（実物画像で十分）
- 122社全員のヒーロー画像（回答薄い社のみ補完）
- 完全自動デプロイ（半自動で十分）
- API課金（ChatGPT Plus + Claude Max契約内で完結）
- Cloudflare Access
