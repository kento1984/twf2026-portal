# TWF2026 みどころポータル スタイルガイド

> Claude Codeが実装時に参照するブランドトークン定義書。
> 値は全て**実装可能な粒度**。Tailwind config / CSS変数 にそのまま落とせる。

---

## 1. ブランドの核

| 要素 | 定義 |
|---|---|
| 世界観 | **産業 × 祭り × 未来感**（溶接の重厚さに、フェスのワクワク感） |
| トーン | プロ仕様、誠実、ちょっと攻めた装飾 |
| 設計哲学 | **素材 > レイアウト**。装飾はPNG透過素材で乗せる。CSSは構造に徹する |
| やってはいけない | 過度なグラデCSS、ふわっとした影、トレンディなglassmorphism、AI感ある中央寄せヒーロー |

## 2. カラーパレット

### コアブランドカラー

| 役割 | 名前 | HEX | 用途 |
|---|---|---|---|
| プライマリ | `corporate-red` | `#C8102E` | タイトル下線、社名ロゴ、強調 |
| セカンダリ | `cta-blue` | `#0066CC` | ボタン、リンク、ナビ |
| アクセント | `twf-orange` | `#FF6B35` | ステージバナー、お祭り装飾、新着 |
| ベース暗 | `gunmetal` | `#2C2C2C` | フッター、強コントラスト背景 |

### スケール（実装用、CSS変数）

```css
:root {
  /* Red */
  --red-50:  #FEF2F4;
  --red-500: #C8102E;  /* corporate-red */
  --red-700: #8B0B20;

  /* Blue */
  --blue-50:  #EFF6FF;
  --blue-500: #0066CC;  /* cta-blue */
  --blue-700: #004999;

  /* Orange */
  --orange-50:  #FFF4EF;
  --orange-500: #FF6B35;  /* twf-orange */
  --orange-700: #C84A1A;

  /* Neutral */
  --gray-50:  #FAFAFA;
  --gray-100: #F4F4F5;
  --gray-200: #E4E4E7;
  --gray-400: #A1A1AA;
  --gray-600: #52525B;
  --gray-800: #2C2C2C;  /* gunmetal */
  --gray-900: #18181B;
}
```

### 配色ルール

- **メイン文字色**: `--gray-800` (`#2C2C2C`)、純黒は使わない
- **背景**: 基本 `#FFFFFF` or `--gray-50`
- **赤の使用は限定的**: タイトル下線、CTA強調、緊急バッジのみ。広面積禁止
- **青はCTA系で統一**: ボタンの「詳細を見る」「PDFを見る」等は全部 `cta-blue`
- **オレンジは祭り装飾**: 注目企画バッジ、NEWバッジ、カウントダウン強調

## 3. タイポグラフィ

### フォントファミリー

```css
font-family: 
  "Noto Sans JP",
  -apple-system, BlinkMacSystemFont,
  "Hiragino Kaku Gothic ProN", 
  "Meiryo", 
  sans-serif;
```

数字専用（カウントダウン等）:
```css
font-family: "Roboto Mono", "SFMono-Regular", monospace;
font-variant-numeric: tabular-nums;
```

### スケール（modular, 1.25 ratio）

| トークン | サイズ | line-height | 用途 |
|---|---|---|---|
| `text-xs` | 12px | 1.5 | キャプション、メタ情報 |
| `text-sm` | 14px | 1.6 | 補助テキスト、バッジ |
| `text-base` | 16px | 1.7 | 本文 |
| `text-lg` | 18px | 1.6 | リード文 |
| `text-xl` | 20px | 1.5 | カードタイトル |
| `text-2xl` | 24px | 1.4 | セクションタイトル |
| `text-3xl` | 30px | 1.3 | ページタイトル |
| `text-4xl` | 38px | 1.2 | ヒーロー見出し |
| `text-5xl` | 48px | 1.1 | カウントダウン数値 |

### ウェイト

- `400` 本文、補助
- `500` 強調、ナビ
- `700` タイトル、ボタンラベル
- `900` ヒーロー、カウントダウン

## 4. 装飾ルール（公式サイトから継承）

### 赤の二重下線（タイトル装飾の定番）

```css
.title-underline {
  display: inline-block;
  padding-bottom: 8px;
  border-bottom: 4px double var(--red-500);
}
```

セクションタイトル（`text-2xl`以上）には**原則これを付ける**。
ページの装飾の柱なので、流用を躊躇しない。

### 「祭り」感の出し方

- ヒーロー周りに**透過PNG装飾**（紙吹雪、リボン、スパーク等）を散らす
- CSSアニメーションは控えめ（floatingの上下動 or fadeIn程度）
- 「祭り」感は**素材**で出す。CSS技法では出さない

## 6. レイアウトトークン

### ボーダー半径
- `rounded-sm`: 4px（バッジ）
- `rounded-md`: 8px（ボタン、入力欄）
- `rounded-lg`: 12px（カード）
- `rounded-xl`: 16px（モーダル、ヒーロー）

### 影
```css
--shadow-sm:   0 1px 2px rgba(0,0,0,0.05);
--shadow-md:   0 4px 6px rgba(0,0,0,0.07), 0 2px 4px rgba(0,0,0,0.04);
--shadow-lg:   0 10px 15px rgba(0,0,0,0.08), 0 4px 6px rgba(0,0,0,0.04);
--shadow-card-hover: 0 12px 24px rgba(0,0,0,0.12);
```

ふわっとした拡散影は使わない。**輪郭がはっきりする影**を選ぶ（産業ツールの硬さに合わせる）。

### ブレイクポイント
- `sm`: 640px（営業現場のスマホ）
- `md`: 768px（タブレット）
- `lg`: 1024px（社内PC）
- `xl`: 1280px（外部PC、メイン）

**メインターゲット**: スマホ（顧客同行時に見る）と PC（事前準備時）の2点。タブレットは中間で吸収。

## 7. コンポーネントトークン

### 7-1. ステータスバッジ（去年Notionから継承＆視覚化）

| ステータス | 色 | アイコン | 用途 |
|---|---|---|---|
| 最優先 | `red-500` 背景・白文字 | 🔥 | 絶対見て欲しい |
| 限定特典あり | `orange-500` 背景・白文字 | 🎁 | TWFのみ特典 |
| 特別割引あり | `red-500` 文字・赤枠 | 💰 | セール価格 |
| 要チェック | `blue-500` 背景・白文字 | ⭐ | 注目案件 |
| 参考情報 | `gray-200` 背景・`gray-800`文字 | ℹ️ | 普通の出展 |
| NEW | `orange-500` 背景・白文字 | ✨ | 新規追加から1週間 |

形状: `rounded-sm`、padding `4px 8px`、`text-sm` `font-weight:700`。
**1カードに最大2個まで**。並べすぎはノイズ。

### 7-2. ボタン

| 種類 | 背景 | 文字 | 用途 |
|---|---|---|---|
| Primary | `cta-blue` | 白 | 「詳細を見る」「PDFを見る」 |
| Secondary | 白 | `cta-blue`、青枠 | 「カテゴリ別に見る」 |
| Danger | `red-500` | 白 | 「最優先のみ表示」 |
| Ghost | 透過 | `gray-800` | テキストリンク的 |

ホバー: 背景10%濃く + `--shadow-md`。トランジション `150ms ease-out`。

### 7-3. メーカーカード（一覧用）

```
┌─────────────────────────┐
│ [ヒーロー画像 16:9]      │
│ ┌───┐ ┌───┐             │
│ │🔥 │ │🎁 │ ← バッジ    │
│ └───┘ └───┘             │
├─────────────────────────┤
│ メーカー名 (text-xl/700)│
│ カテゴリ (text-sm/gray) │
│                         │
│ 提案フック1行           │
│ (text-sm 2行まで省略)   │
│                         │
│ 📍 ブース番号           │
└─────────────────────────┘
```

- `rounded-lg`、`shadow-md`、ホバーで `shadow-card-hover` + 4px上に浮く
- 画像未取得は `--gray-100` プレースホルダー
- カードクリックで詳細遷移（カード全体がクリッカブル）

### 7-4. 検索バー

- 幅100%（max 600px）
- 左にアイコン、右にクリアボタン
- `rounded-md`、`border` `gray-200`、フォーカス時 `cta-blue`枠
- Fuse.js でリアルタイム絞込（メーカー名・読み・カテゴリ・キーワード）

### 7-5. PDFインラインプレビュー

- `<iframe>` で表示（`width:100%; height: 80vh; max-height: 800px`）
- 上にDLボタン（`Primary`スタイル）
- スマホでは「PDFを開く」リンクに切替（iframe重い）

### 7-6. カウントダウン

- 数値: `text-5xl/900` Roboto Mono、`tabular-nums`
- ラベル「日」「時」「分」「秒」: `text-sm/500`
- 背景: `gunmetal`、文字: 白、アクセント数字部分のみ `twf-orange`
- 残り3日切ったら背景フラッシュ（2秒に1回ふわっと光る）

## 9. アニメーション

| 種類 | duration | easing | 用途 |
|---|---|---|---|
| ホバー | 150ms | `ease-out` | ボタン、カード |
| ページ遷移 | 200ms | `ease-in-out` | フェード |
| カウントダウン更新 | なし | - | パッと切替 |
| カード入場 | 300ms | `ease-out` | スクロール時 fadeIn + slideUp 8px |

控えめ。営業ツールなので**速く見える**ことを優先。アニメーションでもったいぶらない。
